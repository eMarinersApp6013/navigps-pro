# SeaGPS Pro — Maritime Position System

Live at: https://marinersapp.com/gps

Standalone maritime GPS web app for bridge officers.
Works on mobile (iOS/Android) and PC browsers.

## Features
- Live GPS position with accuracy monitoring
- Nautical chart (OpenSeaMap)
- GNSS satellite skyplot
- Celestial navigation view (stars, planets, sun, moon)
- Position sharing between devices (6-digit code)
- Night mode (red on black)
- WMM2025 magnetic variation
- GPS spoofing detection (position compare)
